# budget-manager
Projekt na zajęcia z programowania obiektowego

#Wymagania funkcjonalne
-Rejestracja i logowanie użytkownika
-Zarządzanie profilem użytkownika(edycja danych: imię, nazwisko, waluta, miesięczny limit budżetu)
-Dodawanie i edycja transakcji
-Kategorie i podkategorie
-Konta / portfele(np. gotówka, karta X, konto oszczędnościowe saldo konta (równanie sald po transakcjach))
-Budżety i limity
-Raporty i podsumowania(miesięczne i roczne raporty: suma wydatków/przychodów, saldo, prosty wykres danych (dane przygotowane do wyświetlenia))
