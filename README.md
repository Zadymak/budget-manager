# budget-manager
Projekt na zajęcia z programowania obiektowego

# Wymagania funkcjonalne
-Rejestracja i logowanie użytkownika <br>
-Zarządzanie profilem użytkownika(edycja danych: imię, nazwisko, waluta, miesięczny limit budżetu) <br>
-Dodawanie i edycja transakcji <br>
-Kategorie i podkategorie <br>
-Konta / portfele(np. gotówka, karta X, konto oszczędnościowe saldo konta (równanie sald po transakcjach)) <br>
-Budżety i limity <br>
-Raporty i podsumowania(miesięczne i roczne raporty: suma wydatków/przychodów, saldo, prosty wykres danych (dane przygotowane do wyświetlenia)) <br>
