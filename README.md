# budget-manager
Projekt na zajęcia z programowania obiektowego

## Wymagania funkcjonalne
- Korzystanie z aplikacji przez jednego użytkownika. 
- Ustawienie podstawowych danyn użytkownika (nazwa użytkownika, waluta, saldo początkowe).
- Dodawanie przychodu.
- Dodawanie wydatku.
- Każda transkcja zawiera: kwotę, datę, kategorię oraz opis (opcjonalne).
- Wybór kategorii transakcji z predefiniowanej listy (np. jedzenie, paliwo, inne).
- Aplikacja oblicza aktualne saldo na podstawei wszystkich transakcji.
- Aplikacja aktualizuje saldo po każdej dodanej lub usuniętej transakcji.
- Możliwość wyświetlanie podsumowania miesięcznego
  - Podsumowanie zawiera sumę przychodów, sumę wydatków oraz saldo i listę transkacji z wybranego miesiąca.
